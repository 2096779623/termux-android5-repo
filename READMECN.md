## 用法：
1. pkg install git -y
2. git clone https://github.com/2096779623/termux-android5-repo.git
3. cd termux-android5-repo
4. chmod 777 -R *
5. 移动目录下的所有文件到$PREFIX/etc/apt
6. pkg update -y
7. 选N！！！
8.重启termux
