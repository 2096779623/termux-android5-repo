## 用法：
1. git clone https://github.com/2096779623/termux-android5-repo.git
2. cd termux-android5-repo
3. chmod 777 -R *
4. 移动目录下的所有文件到$PREFIX/etc/apt
5. pkg update -y
6. 选N！！！
